So after, what, 8+ months of the same old wordpress blog with automated twitter postings, it was time to move on and begin with a fresh state in mind. I decided that it was to be a complete refresh of the entire raynorpat.com domain, starting with the blog and landing page. I never liked the way Wordpress really flowed and I sure as hell wasn't interested in making a complete theme to make it flow, so I bought myself a Ruby on Rails book, forked a copy of one of the simplest looking homepages (toolmantim's on github), and went to work.

## Dreamhost

I'd first like to take a moment and mention what a PITA this first was… My host is Dreamhost, which is pretty p.i.m.p. for what it costs me per year. I'd recommend them in a heart-beat for some poor student like me. That being said, they really need to work on getting Ruby on Rails updated and some more updated gems. No Capistrano, come on guys… :\

## Blogging… simplified

Anyway's after dealing with my frustrations of my webhost and updating toolmantim's code a tad bit and removing a few peieces of code here and there, I now had a perfect little mini-blog, portfolio, and showcase website for myself that is easy to deploy, maintain, and even update, although ssh'ing into your domain and doing a git pull; cap deploy and typing in 3 passwords is sort of annoying, it's better then Wordpress, that's for sure.

### End

I'd like to thank Tim Lucas (toolmantim) for the orginal code he published on GitHub. Don't worry about your images and sass scripts, they'll be changing a lot in the coming days…

Getting this sucker to run, was pretty fun and challenging and really got me set for some cool updates and other stuff to do. I really got my feet wet so far, so I'd like to give a shoutout to the RoR's team for maintaining an awesome web framework. I'm definately moving my Vapor backend to Rails.
